# 📦 ECR 관련 파일 아카이브

## 🎯 아카이브 이유
프로젝트에서 **퍼블릭 레지스트리**(`ghcr.io/ctfd/ctfd:latest`)를 사용하기로 결정하여, ECR 관련 파일들이 더 이상 필요하지 않게 되었습니다.

## 📋 아카이브된 파일 목록
- `ecr.yaml`: ECR 리포지토리 CloudFormation 템플릿
- `03-setup-ecr.sh`: ECR 리포지토리 생성 스크립트
- `04-setup-team-ecr-role.sh`: 팀원 ECR 접근용 IAM 역할 생성 스크립트

## ⚠️ 참고사항
- 이 파일들은 **참고용으로 보관**됩니다
- 향후 ECR이 다시 필요할 경우 이 파일들을 참고할 수 있습니다
- 현재 프로젝트는 퍼블릭 레지스트리를 사용하므로 이 파일들은 사용되지 않습니다

## 🔄 복원 방법
만약 ECR을 다시 사용하고 싶다면:
1. `archive/ecr/` 디렉토리에서 파일들을 원래 위치로 복원
2. `plan.md`와 `tech.md`에서 ECR 관련 내용 확인
3. `config.sh`에서 ECR 관련 변수 주석 해제
